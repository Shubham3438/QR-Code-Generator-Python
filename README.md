# QR Code Generator (Python)

## Installation require (Using Terminal)
pip install qrcode[pil]

A simple command-line tool to generate scannable QR codes from text or URLs and save them as PNG images.

## Features
- Convert any text or URL into a QR code.
- Save QR as PNG with custom filename, size, and border.
- Easy to run locally; suitable for sharing links, contact info, or event passes.
